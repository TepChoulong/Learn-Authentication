# Welcome to Learn-Authentication

In this repository, I will guide you through the process of creating and learning how to build an authentication system for your application. Furthermore, this project is free for everyone to use for learning or production without spending time on developing it from scratch.

## Tech Stacks

These are the tech stacks that I am going to use in this project.

### Frontend

- ReactJs
- Zod
- Axois

### Backend

- ExpressJs
- JWT (JsonWebToken)

### Database

- Postgresql
